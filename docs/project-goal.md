# Project Goal

Build an NYC real estate operations and research workspace that:

- tracks listings and property metadata
- summarizes market inventory and status changes
- supports page-level dashboard planning
- keeps API, data, and UI contracts aligned through DuckFlow

Initial success means:

- docs, schema, API contracts, and page plans can all be loaded into DuckFlow
- local API smoke tests run from the API tester
- dashboard wireframe plans stay linked to saved tests
