# Agent Source Mirror

This directory is the editable source mirror for `.agents/`.

In this starter project:

- `agents/` is the writable source-of-truth
- `.agents/` is the compatibility copy for tools that expect the hidden directory name

If you move this project to its final repo, you can:

1. keep both
2. rename `agents/` to `.agents/`
3. or periodically sync `agents/` into `.agents/`
